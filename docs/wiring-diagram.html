// Pin Connection Reference

From Pin To Pin Wire Color Notes
PN532 SDA → ESP32 A4 (D18) Blue I2C Data — 4.7kΩ pullup to 3.3V
PN532 SCL → ESP32 A5 (D19) Yellow I2C Clock — 4.7kΩ pullup to 3.3V
PN532 VCC → ESP32 3V3 Orange 3.3V — do NOT use 5V
PN532 GND → ESP32 GND Black Common ground
LED Ring DI → ESP32 D5 Purple Data in — direct connection
LED Ring 5V → ESP32 VBUS Red USB 5V rail — no external PSU needed. Cap brightness ~50% on standard USB.
LED Ring DO → — N/C — Data out — unused unless daisy-chaining additional LEDs
LED Ring GND → ESP32 GND Black Common ground
ESP32 USB-C → Host PC USB Green HID keyboard output to TouchDesigner host

⚠ PN532 I2C Mode

The Elechouse v3 has a mode selector switch. For I2C set: SCL=HIGH, SDA=LOW (switch positions 1=OFF, 2=ON). Do this before powering up. I2C address will be 0x24.

⚡ Power Budget

40 LEDs at full white = ~2.5A at 5V. The LED ring is powered via ESP32 VBUS (USB 5V rail) — no external PSU needed. Cap brightness ~50% on standard USB ports. For full brightness, use a USB-C adapter rated 2A+.

🔌 Pull-up Resistors

Add 4.7kΩ resistors from SDA → 3.3V and SCL → 3.3V. The Elechouse module often includes these on-board — check before adding.
